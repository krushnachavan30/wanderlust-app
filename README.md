# Wanderlust

**Technologies:** JavaScript, HTML/CSS, Bootstrap, Express.js, Node.js, MongoDB, Mapbox API  

**Description:**  
Wanderlust is a full-stack, immersive, and user-friendly travel web application designed to inspire and assist travel enthusiasts in planning their next adventure.  

**Features:**  
- Interactive maps with Mapbox for destinations, nearby attractions, and routes  
- User authentication and management  
- Listing of travel locations with details and images  
- Responsive UI with Bootstrap  

**Deployment:**  
- Deployed on Render for optimal performance and reliability
